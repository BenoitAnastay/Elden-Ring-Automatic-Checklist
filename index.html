Save manager

Welcome! I'm CyberGiant this site is made to analyze the inventory of an elden ring save to automatically compile a checklist.

To find Elden Ring save:
You will find your Elden Ring save file in the your user directory. Right-click the folder icon on your quick bar and click 'File Explorer'. Then select 'This PC' from the menu on the left of the window and click through the director until you reach this location:
C:\Users\[username]\AppData\Roaming\EldenRing.

Once here, you should see a file with a bunch of random-looking numbers. That number is your 64-bit Steam ID, in case you were wondering.
Click into this folder and you should see a file or multiple that end in .sl2. This is your save file.

This site does not alter Elden Ring saves in any way, as the file is opened in read-only mode.
